## Claude Code Cheatsheet

keyboard shortcuts၊ commands၊ MCP servers၊ memory management၊ workflows နှင့် best practices များကို လွှမ်းခြုံပါဝင်သည့် ပြည့်စုံသော လမ်းညွှန်များဖြင့် Claude Code ကို ကျွမ်းကျင်အောင် လေ့လာပါ။

### ⌨️ Keyboard Shortcuts

**အမြန်ကြည့်ရှုရန် (Quick Reference)**

Mac နှင့် Windows အတွက် မရှိမဖြစ်လိုအပ်သော keyboard shortcuts များကို ကျွမ်းကျင်အောင် လေ့လာပါ။ အခြေခံ navigation၊ mode switching နှင့် input shortcuts များဖြင့် သင့်အလုပ်လုပ်ပုံကို မြန်ဆန်စေပါ။

- အခြေခံ shortcuts များ (Cancel, Exit, Clear)
- Mode switching (Permission, Model, Thinking)
- Input methods နှင့် prefixes များ
- Navigation နှင့် search commands များ

[View All Shortcuts](https://claude-guides.com/#keyboard-shortcuts)

### / Slash Commands

**စွမ်းအားမြင့် ကိရိယာများ (Power Tools)**

slash commands များဖြင့် စွမ်းအားမြင့် လုပ်ဆောင်ချက်များကို အကောင်အထည်ဖော်ပါ။ tasks၊ files၊ configuration နှင့် အခြားအရာများကို သင့် terminal မှ တိုက်ရိုက် စီမံခန့်ခွဲပါ။

- Task management (/clear, /todo)
- File operations (/ls, /cat, /git)
- Configuration နှင့် tools (/settings, /help)
- AI operations (/ask, /improve, /test)

[Explore Commands](https://claude-guides.com/#slash-commands)

### 🔗 MCP Servers

**ပေါင်းစပ်ချိတ်ဆက်မှုများ (Integrations)**

ပြင်ပ ဝန်ဆောင်မှုများနှင့် tools များနှင့် ချိတ်ဆက်ပါ။ MCP servers များသည် GitHub၊ databases နှင့် အခြားပေါင်းစပ်ချိတ်ဆက်မှုများဖြင့် Claude Code ၏ လုပ်ဆောင်နိုင်စွမ်းကို ချဲ့ထွင်ပေးသည်။

- GitHub integration နှင့် workflows များ
- Database နှင့် file system အသုံးပြုခွင့်
- Web APIs နှင့် ပြင်ပ ဝန်ဆောင်မှုများ
- Custom MCP server တပ်ဆင်ခြင်း

[Learn Integration](https://claude-guides.com/#mcp-servers)

### 💾 Memory Management

**ပိုမိုကောင်းမွန်အောင် ပြုလုပ်ခြင်း (Optimization)**

Claude Code က context နှင့် memory ကို မည်သို့ စီမံခန့်ခွဲသည်ကို နားလည်ပါ။ ပိုရှည်သော sessions များနှင့် ရှုပ်ထွေးသော projects များအတွက် ပိုမိုကောင်းမွန်အောင် ပြုလုပ်နည်းကို လေ့လာပါ။

- Context window management
- Memory persistence နှင့် projects များ
- Token usage optimization
- Session persistence

[Read Full Guide](https://claude-guides.com/memory-management.html)

### ⚙️ Configuration & Settings

**စိတ်ကြိုက်ပြင်ဆင်ခြင်း (Customization)**

Claude Code ကို သင့်အလုပ်လုပ်ပုံနှင့် ကိုက်ညီအောင် စိတ်ကြိုက်ပြင်ဆင်ပါ။ settings၊ hooks၊ preferences နှင့် environment variables များကို configure လုပ်ပါ။

- Settings.json configuration
- Shell hooks နှင့် automation
- Environment variables တပ်ဆင်ခြင်း
- IDE integrations နှင့် extensions များ

[Setup Guide](https://claude-guides.com/blog/best-practices.html)

### 🚀 Advanced Workflows

**Best Practices**

အဆင့်မြင့် ပုံစံများနှင့် workflows များကို ရှာဖွေတွေ့ရှိပါ။ debugging၊ testing၊ code review နှင့် ရှုပ်ထွေးသော projects များအတွက် ကျွမ်းကျင်သူ နည်းဗျူဟာများကို လေ့လာပါ။

- Debugging နည်းစနစ်များနှင့် tools များ
- Testing နှင့် CI/CD integration
- Code review workflows
- Multi-file project management

[Explore Workflows](https://claude-guides.com/advanced-workflows.html)

### 🎯 Prompt Engineering

**ကျွမ်းကျင်မှုများ (Skills)**

ထိရောက်သော prompts များ ရေးဆွဲခြင်း အတတ်ပညာကို ကျွမ်းကျင်အောင် လေ့လာပါ။ မတူညီသော အခြေအနေများတွင် Claude AI ဖြင့် ပိုမိုကောင်းမွန်သော ရလဒ်များ ရရှိရန် နည်းစနစ်များကို လေ့လာပါ။

- Prompt ၏ ဖွဲ့စည်းပုံနှင့် ရှင်းလင်းမှု
- Context နှင့် ဥပမာများ
- Role-based prompting
- ထပ်ခါတလဲလဲ ပြုပြင်မွမ်းမံခြင်း

[Learn Techniques](https://claude-guides.com/prompt-engineering.html)

**ကျွမ်းကျင်သူ အကြံပြုချက်များ (Pro Tips)**

သိပ်မသိကြသော အင်္ဂါရပ်များနှင့် power-user အကြံပြုချက်များကို ဖော်ထုတ်ပါ။ အဆင့်မြင့် နည်းစနစ်များဖြင့် သင့် Claude Code ကျွမ်းကျင်မှုကို မြှင့်တင်ပါ။

- စာရွက်စာတမ်းမှ မှတ်တမ်းမတင်ထားသော အင်္ဂါရပ်များ
- Keyboard modifier ပေါင်းစပ်မှုများ
- Command chaining နည်းစနစ်များ
- Performance လျှို့ဝှက်ချက်များ
