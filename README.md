# vibe-code-tours CLI

`npx vibe-code-tours` — student + instructor toolkit for the Vibe Code Tours bootcamp.

Status: scaffold. See `PLAN.md` for the full phased plan.

## Quick start (when published)

```bash
npx vibe-code-tours              # welcome + cohort status
npx vibe-code-tours --cohort 2 --free   # enrol intent
npx vibe-code-tours doctor       # env diagnostic
npx vibe-code-tours setup        # student env setup
npx vibe-code-tours api-setup    # AI key wizard
```
